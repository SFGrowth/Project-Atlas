# Observer State Definitions

## Purpose

This document defines the market-state terms used by Atlas Observer v0.1.

The goal is to make every state observable, testable, and reviewable.

Atlas should not produce vague labels.

Every state should help answer:

> Is this environment worth trading, or should risk be reduced?
